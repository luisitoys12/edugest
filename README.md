# EduGest — Gestión Escolar SaaS

Plataforma SaaS para gestión escolar completa: calificaciones, grupos, alumnos, boletas PDF, reportes analíticos y portal para padres.

## 🌐 Demo en vivo
https://luisitoys12.github.io/edugest/

## ✨ Funciones
- Dashboard con KPIs en tiempo real
- Gestión de alumnos, docentes y grupos
- Calificaciones por materia y periodo
- Asistencias
- **Boletas PDF** — vista previa y descarga
- **Reportes analíticos** — 6 gráficas interactivas
- **Portal para Padres** — acceso por código único de alumno (ej: `BJ-0001`)
- Comunicados y avisos
- Planes y precios

## 🏗️ Stack
- **Frontend**: React 18 + Vite + Tailwind CSS + shadcn/ui
- **Backend**: Express 5 + TypeScript
- **ORM**: Drizzle (listo para Supabase)
- **Deploy**: GitHub Pages (frontend) + GitHub Actions (CI/CD)

## 🚀 Desarrollo local
```bash
npm install
npm run dev
```

## 🗄️ Conectar Supabase
Ver [SUPABASE.md](./SUPABASE.md) para instrucciones completas.

## 🔑 Códigos de alumno
Formato: `BJ-0001` hasta `BJ-0050`. Los padres los usan en el Portal para Padres.
